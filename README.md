# Getting Started

## Prerequisites

- [Astro](https://docs.astro.build/en/getting-started/)

## Install the dependencies on the project

```bash
npm install
```

## Run the project

```bash
npm run dev
```

### Open the code and start customizing

Your site is now running at http://localhost:4321!
